SmartMenus v6.0.2

Release Date: March 15, 2008
An advanced, cross-browser, simple-to-use and accessible list-based DHTML Menu / JavaScript Menu
©2008 Vadikom - Bulgarian company settled in Plovdiv

Installation

Please take a look at the "1 Getting started" section in the SmartMenus 6 User's Manual (the "manual.html" file in this directory) for a quick start.

License

The SmartMenus script is free for non-commercial use but is not free software. For details on licensing, please visit http://www.smartmenus.org/license/.

Some key features

  • Accessible to all web browsers and assistive technology
  • List-based (search engine friendly) markup - can be easily generated from any kind of database
  • Section 508 compliant
  • Completely CSS driven
  • Controllable delay (timeout) before showing/hiding any sub menu
  • Allows the creation of unlimited number of menu trees on the same page
  • Unlimited number of sub menu levels supported
  • Horizontal or vertical main menu items arrangement
  • Absolute/relative/fixed positioning for the main menus supported
  • Right-to-left and bottom-to-top display of the sub menus is possible
  • Full support for RTL text/pages (e.g. Hebrew, Arabic)
  • Full window size detection in all supported browsers - the menus will always be kept inside the window's boundaries
  • Option for automatically adjustable width for the sub menus is available (including min/max settings)
  • Tab key navigation friendly
  • Fully customizable menu styles
  • Compact and optimized code

Browser support

Fully supported browsers

The following browsers are fully supported by the SmartMenus 6 script.

  • IE 5+ on Windows
  • Netscape 6+ (and all Mozilla/Gecko powered browsers like Firefox)
  • Safari 1+
  • IE 5.1 and IE 5.2 on Macintosh
  • Opera 7+
  • Konqueror 3.2+

Partially supported browsers

The following browsers have pretty decent CSS support but due to lack of good DOM support (or some crucial CSS bugs) are not fully supported. In these browsers the main menu will be styled as in the fully supported browsers but no sub menus will appear.

  • iCab 3
  • IE 5.0 on Macintosh
  • Opera 5/6
  • Konqueror 3.0.x and 3.1.x

Other browsers

Due to the fact that the SmartMenus 6 menu definitions are embedded on the page in the form of list-based markup, the links in the menus are accessible to all other browsers not listed above (like search engines, small-screen device browsers, text-mode browsers) and assistive technology (like screen readers).
